# 🌟 About This Portfolio

Welcome to my **Data Analytics Internship Portfolio**, developed during the **60-Day Data Analytics Internship Program** conducted by **ApexPlanet Software Pvt. Ltd.**

This repository represents my complete learning journey through real-world data analytics projects. Throughout this internship, I worked on every stage of the **Data Analytics Lifecycle**, from collecting and cleaning raw datasets to building interactive dashboards, validating insights using statistical analysis, and presenting business recommendations through professional storytelling.

Rather than being a collection of individual tasks, this repository demonstrates a structured, end-to-end analytics workflow similar to projects carried out by professional data analysts in the industry.
